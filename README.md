# WebAn Engine

Движок для созданий игр на основе python и библиотеки Pygame![enter image description here](https://github.com/evembar/WebAn-Engine/raw/main/hd-logo2.png)


## Структура движка

Движок является модульным, тем самым не делает его перегруженным
Основными являются load.py, boot.py и engine.py. Start.py и game.py являются основными для построения игры.

## Команды движка

### Ver. 0.01
Для отображения логотипа студии или себя используются 3 функции: 
ANIATE() - рисует динамичный логотип
SPLASH() - рисует статичный логотип
VIDEO() - воспроизводит видео
